Alke Wallet 

Billetera Virtual desarrollada por Kevin Morillas.
Cuenta con sistema que emula el login, un menu que muestra el saldo y puede redirigir a las demas funcionalidades
como depositar, transferir y ver el historial de transacciones.

*¡Prueba la aplicacion en vivo!*
https://izukqq.github.io/alke-wallet/

Como la pagina de Login.html simula un login, no es necesario recibir credenciales, ya que los datos que almacena
son en sessionStorage, pero si es obligatorio logear con formato correo@email.com y una contraseña > 6 caracteres.

Tecnologias: HTML5, CSS3 (Bootstrap + Flexbox + Custom), JavaScript (JQuery), Git.
